## Features
- User Signup & Login
- Create text & image posts
- Like & comment on posts
- Search posts by username
- Persistent data storage
- Responsive social feed UI

## Deployment
- Frontend: Netlify
- Backend: Render
- Database: MongoDB Atlas

## How to run locally
### Backend
cd backend  
npm install  
npm start  

### Frontend
cd frontend  
npm install  
npx expo start
